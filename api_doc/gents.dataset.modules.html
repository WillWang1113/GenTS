class gents.dataset.modules.Physionet(agg_minutes: int = 60, select_seq_dim: List[int | str] | None = None, batch_size: int = 32, data_dir: str = './data', condition: str | None = None, scale: bool = True, inference_batch_size: int = 1024, max_time: float = 1.0, add_coeffs: str | None = None, **kwargs)

Bases: BaseDataModule

Physionet Challenge 2012 Dataset

For each patient measurement, the original data is irregularly recorded for 48 hours. For each record, the time format is “HH:MM”. Therefore, it can be treated as a long sequence at 1-min resolution, and for 48 hours, the total time steps are 48*60. We allow the users to aggreate (resample) to make the sequence shorter by setting agg_minutes, e.g. agg_minutes=60 is resampled at 1-hour level, resulting in sequences with 48 time steps.

In this way, this dataset has missing values at the unrecorded time steps.

Note

Originally has missing values. irregular_dropout is disabled.
